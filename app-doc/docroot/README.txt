#简介
 
 这个程序是用于方便收集资料，具体是什么，也许是个简单的Blog，也许是个人或团队的知识收集工具。
##程序简介

 这个程序是用PHP写的，带了一个修订后的Markdown解析库。文档管理完全依赖与百度云（BAE）平台的SVN。

 * 在目录下以txt 结尾的都会被认为是 markdown 文件，然后被解析为Html
 * 图片文件可以正常浏览，其他格式的文件可以正常访问，当然包括 html 、css什么的。
 * 访问路径：当前目录就是网站根目录
 * 在目录下以 _ 下划线开头的文件或者目录是隐藏目录，半公开状态
 
###运行环境
 * PHP
 * BAE云平台
